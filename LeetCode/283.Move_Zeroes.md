# 0283. Move Zeroes

## Description

Given an integer array `nums`, move all `0` values to the end of the array while maintaining the relative order of the non-zero elements.

The operation must be performed **in-place** without creating a copy of the array.

---

## Example

### Example 1

**Input**

```text
nums = [0,1,0,3,12]
```

**Output**

```text
[1,3,12,0,0]
```

### Example 2

**Input**

```text
nums = [0]
```

**Output**

```text
[0]
```

---

# My Solution

```java
class Solution {
    public void moveZeroes(int[] nums) {

        int k = 0;

        for (int i = 0; i < nums.length; i++) {
            if (nums[i] != 0) {
                nums[k] = nums[i];
                k++;
            }
        }

        while (k < nums.length) {
            nums[k] = 0;
            k++;
        }
    }
}
```

---

# Approach

這題可以分成兩個階段處理：

1. 將所有非零元素依照原本順序移動到陣列前方。
2. 將剩餘的位置全部補成 `0`。

使用變數：

```java
int k = 0;
```

`k` 代表下一個非零元素應該放入的位置。

---

## Step 1：移動非零元素

```java
for (int i = 0; i < nums.length; i++) {
    if (nums[i] != 0) {
        nums[k] = nums[i];
        k++;
    }
}
```

使用 `i` 掃描整個陣列。

當：

```java
nums[i] != 0
```

代表目前找到一個非零元素，因此將它放到索引 `k`：

```java
nums[k] = nums[i];
```

接著：

```java
k++;
```

讓 `k` 移動到下一個可以放置非零元素的位置。

---

## Step 2：將剩餘位置補成零

```java
while (k < nums.length) {
    nums[k] = 0;
    k++;
}
```

第一個迴圈結束後，所有非零元素都已經被移動到陣列前面。

此時 `k` 會停在第一個應該放置 `0` 的位置。

所以只需要從 `k` 開始，將剩餘的陣列位置全部設成 `0`。
