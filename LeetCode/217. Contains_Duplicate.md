# 0217. Contains Duplicate

## Description

Given an integer array `nums`, return `true` if any value appears at least twice in the array, and return `false` if every element is distinct.

---

## Example

### Example 1

**Input**

```text
nums = [1,2,3,1]
```

**Output**

```text
true
```

---

### Example 2

**Input**

```text
nums = [1,2,3,4]
```

**Output**

```text
false
```

---

### Example 3

**Input**

```text
nums = [1,1,1,3,3,4,3,2,4,2]
```

**Output**

```text
true
```

---

#  My Solution

## Solution 1 - Brute Force

```java
class Solution {
    public boolean containsDuplicate(int[] nums) {

        for (int i = 0; i < nums.length; i++) {

            for (int j = i + 1; j < nums.length; j++) {

                if (nums[i] == nums[j]) {
                    return true;
                }

            }

        }

        return false;
    }
}
```

---

## Solution 2 - HashSet

```java
class Solution {
    public boolean containsDuplicate(int[] nums) {

        Set<Integer> seen = new HashSet<>();

        for (int i = 0; i < nums.length; i++) {

            if (seen.contains(nums[i])) {
                return true;
            }

            seen.add(nums[i]);
        }

        return false;
    }
}
```

先使用暴力`for...for`解法，雖然可以正確找出重複元素，但需要將每個元素與後面的所有元素逐一比較，
如果遇上大量數字很容易會查很久，
因此改用`HashSet`。

利用`HashSet`可以將已經看過的數字記錄起來，每次只需要使用 `contains()` 判斷是否已經存在。
如果已經存在，代表找到重複元素，可以直接回傳 `true`；如果不存在，就使用 `add()` 將目前數字加入 HashSet。
因此可以省略第二層 `for` 的搜尋。

#  Note

## HashSet

```java
Set<Integer> seen = new HashSet<>();
```
HashSet 只儲存元素本身，不會記錄 Index。

```java
seen.add(value)
```
新增元素。

```java
seen.contains(value)
```
判斷元素是否存在。

```java
seen.remove(value)
```
移除元素。

---

## HashSet vs HashMap

| HashSet  | HashMap                |
| -------- | ---------------------- |
| Value    | Key → Value            |
| 判斷元素是否存在 | 紀錄額外資訊（例如 Index、Count） |
