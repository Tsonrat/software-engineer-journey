# 125. Valid Palindrome

## Description

A phrase is a palindrome if, after converting all uppercase letters into lowercase letters and removing all non-alphanumeric characters, it reads the same forward and backward.

Given a string `s`, return `true` if it is a palindrome, or `false` otherwise.

---

## Example

### Example 1

**Input**

```text
s = "A man, a plan, a canal: Panama"
```

**Output**

```text
true
```

---

### Example 2

**Input**

```text
s = "race a car"
```

**Output**

```text
false
```

---

### Example 3

**Input**

```text
s = " "
```

**Output**

```text
true
```

---

#  My Solution

## Solution 1 - ArrayList + Two Pointers

```java
class Solution {
    public boolean isPalindrome(String s) {

        List<Character> list = new ArrayList<>();

        for (int i = 0; i < s.length(); i++) {

            char c = s.charAt(i);

            if (!Character.isLetterOrDigit(c)) {
                continue;
            }

            list.add(Character.toLowerCase(c));
        }

        int left = 0;
        int right = list.size() - 1;

        while (left < right) {

            if (!list.get(left).equals(list.get(right))) {
                return false;
            }

            left++;
            right--;
        }

        return true;
    }
}
```

先遍歷整個字串，利用 `Character.isLetterOrDigit()` 過濾掉所有非英數字元，再透過 `Character.toLowerCase()` 將字元統一轉成小寫，並存入 `ArrayList`。

完成前處理後，使用 **Two Pointers** 分別從陣列的左右兩端開始比較。若兩端字元不同，代表不是回文，立即回傳 `false`；若一路比較至中間皆相同，則代表為回文，最後回傳 `true`。

#  Note

## Palindrome

Palindrome（回文）是指：

> 從左往右讀與從右往左讀完全相同的字串。

例如：

```text
racecar
level
madam
```

本題需要先：

- 忽略所有非英數字元
- 不區分大小寫

例如：

```text
A man, a plan, a canal: Panama
```

處理後會變成：

```text
amanaplanacanalpanama
```

因此結果為 `true`。

---

## Character

### 常用方法

```java
Character.isLetterOrDigit(c);
```

判斷是否為英文字母或數字。

```java
Character.toLowerCase(c);
```

將字元轉為小寫。

---

## Two Pointers

建立兩個指標：

```text
left -----------------> <---------------- right
```

每次比較左右兩端字元：

```java
list.get(left)
list.get(right)
```

若相同：

```java
left++;
right--;
```

若不同：

```java
return false;
```

直到兩個指標相遇為止。
