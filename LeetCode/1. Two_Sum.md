# 0001. Two Sum

> Difficulty: Easy
>
> Tags: Array, HashMap

---

# 📝 Problem

## Description

Given an array of integers `nums` and an integer `target`,
return the indices of the two numbers such that they add up to the target.

You may assume that each input has exactly one solution.

---

## Example

### Example 1

**Input**

```text
nums = [2,7,11,15]
target = 9
```

**Output**

```text
[0,1]
```

**Explanation**

```text
nums[0] + nums[1] == 9
```

---

### Example 2

**Input**

```text
nums = [3,2,4]
target = 6
```

**Output**

```text
[1,2]
```

---

# 💻 My Solution

## Solution 1 - Brute Force

```java
class Solution {
    public int[] twoSum(int[] nums, int target) {

        for (int i = 0; i < nums.length; i++) {

            for (int j = i + 1; j < nums.length; j++) {

                if (nums[i] + nums[j] == target) {
                    return new int[]{i, j};
                }

            }
        }

        return new int[]{};
    }
}

```
## Solution 2 - HashMap

```java
class Solution {
    public int[] twoSum(int[] nums, int target) {
        Map<Integer, Integer> map = new HashMap<>();

        for (int i = 0; i < nums.length; i++){
            int answer = target - nums[i];

            if(map.containsKey(answer)){
                return new int[]{map.get(answer), i};
            }

            map.put(nums[i], i);
        }

        return new int[]{};
    }
}
```
先使用暴力for...for解法，發現效益不好才考慮使用HashMap做為第二解法， \n
其理由為HashMap可以使用map.put()去紀錄當前的key/value， \n
因此可以省略第二個for循環去查找相對應的資料。

# 📖 Note

## HashMap

建立 HashMap：

```java
Map<Integer, Integer> map = new HashMap<>();
```

| Key | Value |
|------|-------|
| nums[i] | index |

常用方法：

```java
map.containsKey(key)
```

判斷 Key 是否存在。

```java
map.get(key)
```

取得 Value。

```java
map.put(key, value)
```

新增或更新資料。

---

## ArrayList vs HashMap

| ArrayList | HashMap |
|-----------|----------|
| Index → Value | Key → Value |
| list.get(index) | map.get(key) |


- HashMap 適合快速查詢。
- containsKey() 用來判斷 Key 是否存在。
- get() 取得 Value。
- put() 新增或更新資料。
- HashMap 可以將 Two Sum 從 O(n²) 優化到 O(n)。
