# 0001. Two Sum

> Difficulty: Easy
>
> Tags: Array, HashMap

---

# 📝 Problem

## Description

Given an array of integers `nums` and an integer `target`,
return the indices of the two numbers such that they add up to the target.

You may assume that each input has exactly one solution.

---

## Example

### Example 1

**Input**

```text
nums = [2,7,11,15]
target = 9
```

**Output**

```text
[0,1]
```

**Explanation**

```text
nums[0] + nums[1] == 9
```

---

### Example 2

**Input**

```text
nums = [3,2,4]
target = 6
```

**Output**

```text
[1,2]
```

# 📖 Note

## HashMap

建立 HashMap：

```java
Map<Integer, Integer> map = new HashMap<>();
```

| Key | Value |
|------|-------|
| nums[i] | index |

常用方法：

```java
map.containsKey(key)
```

判斷 Key 是否存在。

```java
map.get(key)
```

取得 Value。

```java
map.put(key, value)
```

新增或更新資料。

---

## ArrayList vs HashMap

| ArrayList | HashMap |
|-----------|----------|
| Index → Value | Key → Value |
| list.get(index) | map.get(key) |


- HashMap 適合快速查詢。
- containsKey() 用來判斷 Key 是否存在。
- get() 取得 Value。
- put() 新增或更新資料。
- HashMap 可以將 Two Sum 從 O(n²) 優化到 O(n)。
