# 0242. Valid Anagram

## Description

Given two strings `s` and `t`, return `true` if `t` is an anagram of `s`, and `false` otherwise.

An **Anagram** is a word or phrase formed by rearranging the letters of a different word, using all the original letters exactly once.

---

## Example

### Example 1

**Input**

```text
s = "anagram"
t = "nagaram"
```

**Output**

```text
true
```

---

### Example 2

**Input**

```text
s = "rat"
t = "car"
```

**Output**

```text
false
```

---

# 💻 My Solution

## Solution 1 - HashMap

```java
class Solution {
    public boolean isAnagram(String s, String t) {

        if (s.length() != t.length()) {
            return false;
        }

        Map<Character, Integer> map = new HashMap<>();

        for (int i = 0; i < s.length(); i++) {

            char c = s.charAt(i);

            if (map.containsKey(c)) {
                map.put(c, map.get(c) + 1);
            } else {
                map.put(c, 1);
            }

        }

        for (int i = 0; i < t.length(); i++) {

            char c = t.charAt(i);

            if (!map.containsKey(c)) {
                return false;
            }

            map.put(c, map.get(c) - 1);

            if (map.get(c) < 0) {
                return false;
            }

        }

        return true;
    }
}
```

先判斷兩個 String 長度是否一致，如果長度不同，就不可能是 Anagram，因此直接回傳 `false`。
接著建立一個 `HashMap<Character, Integer>`，利用 Key 紀錄字母、Value 紀錄字母出現的次數。
第一個 `for` 先統計第一個字串 `s` 每個字母出現的次數。
第二個 `for` 再掃描第二個字串 `t`：
* 使用 `containsKey()` 判斷該字母是否存在。
* 如果不存在，代表第二個字串出現了第一個字串沒有的字母，直接回傳 `false`。
* 如果存在，就將該字母的次數減一。
* 如果減完小於 `0`，代表第二個字串的該字母數量比第一個字串更多，因此回傳 `false`。
如果全部檢查完成都沒有問題，就代表兩個字串的字母種類與數量完全一致，因此回傳 `true`。

# 📖 Note

## HashMap

```java
Map<Character, Integer> map = new HashMap<>();
```

| Key       | Value |
| --------- | ----- |
| Character | Count |

HashMap 用來記錄每個字母出現的次數。

---

```java
map.put(key, value)
```

新增或更新資料。

```java
map.get(key)
```

取得目前 Value。

```java
map.containsKey(key)
```

判斷 Key 是否存在。

---

* 長度不同，一定不是 Anagram。
* HashMap 適合記錄「Key → Value」的資料。
* 本題利用 **Character → Count** 紀錄每個字母出現次數。
* 第一個 `for` 負責統計字母數量。
* 第二個 `for` 負責逐一扣除字母數量。
* `containsKey()` 用來判斷字母是否存在。
* 如果字母不存在，直接回傳 `false`。
* 如果字母數量扣到小於 `0`，代表第二個字串該字母數量較多，回傳 `false`。
* 全部扣完都沒有問題，即代表兩個字串互為 Anagram。
