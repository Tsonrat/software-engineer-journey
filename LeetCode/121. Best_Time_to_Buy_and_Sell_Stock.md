# 121. Best Time to Buy and Sell Stock

## Description

You are given an array `prices` where `prices[i]` is the price of a given stock on the `i`th day.

You want to maximize your profit by choosing a single day to buy one stock and choosing a different day in the future to sell that stock.

Return the maximum profit you can achieve from this transaction. If you cannot achieve any profit, return `0`.

---

## Example

### Example 1

**Input**

```text
prices = [7,1,5,3,6,4]
```

**Output**

```text
5
```

**Explanation**

Buy on day 2 (price = 1) and sell on day 5 (price = 6).

Profit = `6 - 1 = 5`.

---

### Example 2

**Input**

```text
prices = [7,6,4,3,1]
```

**Output**

```text
0
```

**Explanation**

In this case, no transaction can achieve a positive profit.

---

#  My Solution

## Solution 1 - Greedy

```java
class Solution {
    public int maxProfit(int[] prices) {

        int minPrice = prices[0];
        int maxProfit = 0;

        for (int i = 1; i < prices.length; i++) {

            if (prices[i] < minPrice) {
                minPrice = prices[i];
            }

            int profit = prices[i] - minPrice;

            if (profit > maxProfit) {
                maxProfit = profit;
            }
        }

        return maxProfit;
    }
}
```

使用 **Greedy** 的方式，只需要遍歷陣列一次。

利用 `minPrice` 紀錄目前遍歷過的最低股價，代表最佳買入價格。

每一天都計算：

```text
今天價格 - 目前最低價格
```

得到如果今天賣出的獲利 (`profit`)。

若 `profit` 大於目前最大的獲利 (`maxProfit`)，就更新答案。

如此只需要一次遍歷即可完成。

---

#  Note

## Greedy

Greedy（貪心演算法）是一種：

> 每一步都做出目前看起來最好的選擇。

本題中：

- `minPrice`：目前為止最低的股價（最佳買點）
- `maxProfit`：目前找到的最大獲利


---

## Why start from index 1?

由於：

```java
int minPrice = prices[0];
```

第一天已經作為初始最低價格，因此迴圈直接從第二天開始：
避免重複判斷第一筆資料。

---

## Key Idea

每一天只做兩件事：

### 1. 更新最低價格

```java
if (prices[i] < minPrice) {
    minPrice = prices[i];
}
```

### 2. 計算今天賣出的最大獲利

```java
int profit = prices[i] - minPrice;
```

若：

```java
profit > maxProfit
```

則更新：

```java
maxProfit = profit;
```
